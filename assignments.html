Simulate the following and generate the corresponding plot:

  1. x(t)
  2. x(t − 3)
  3. x(2t)
  4. x(1/2 t)
  5. x(2t + 3)

a) Code for Matlab Function Block to Generate x(t)

function y = fcn(t)
    if t >= 0 && t < 2
        y = t;
    elseif t >= 2 && t <= 6
        y = 2;
    elseif t > 6 && t <= 8
        y = 8 - t;
    else
        y = 0;
    end

function y = fcn(t)
    t1 = 2 * t + 3;

    if t1 >= 0 && t1 < 2
        y = (t1 - 3) / 2;
    elseif t1 >= 2 && t1 <= 6
        y = 2;
    elseif t1 > 6 && t1 <= 8
        y = 8 - t1;
    else
        y = 0;
    end
end

b)

Convolution of x(t) = sin(2πt) and h(t) = t

To find the convolution of x(t) = sin(2π t) for 0 ≤ t ≤ 2 and h(t) = t for 0 ≤ t ≤ 1, we compute:

y(t) = (x * h)(t) = ∫0t x(τ) h(t - τ) dτ

  1. For 0 ≤ t ≤ 1:
    y(t) = ∫0t sin(2π τ) ⋅ (t - τ) dτ
  2. For 1 < t ≤ 2:
    y(t) = ∫t-11 sin(2π τ) ⋅ (t - τ) dτ
  3. For 2 < t ≤ 3:
    y(t) = ∫t-21 sin(2π τ) ⋅ (t - τ) dτ

Matlab code for finding convolution of x(t) and h(t) is ;

Install Symbolic Math Toolbox 

syms t tau;

x_t = sin(2 * pi * tau) * (tau >= 0 & tau < 2);
h_t = tau * (tau >= 0 & tau < 1);

y_t = int(x_t * subs(h_t, tau, t - tau), tau, 0, t);

y_t_func = matlabFunction(y_t, 'Vars', t);

t_vals = linspace(0, 3, 1000);
y_vals = zeros(size(t_vals));

for i = 1:length(t_vals)
    t_val = t_vals(i);
    if t_val < 1
        y_vals(i) = double(int(sin(2 * pi * tau) * tau, tau, 0, t_val));
    elseif t_val < 2
        y_vals(i) = double(int(sin(2 * pi * tau) * tau, tau, 0, 1));
    else
        y_vals(i) = double(int(sin(2 * pi * tau) * tau, tau, 0, 1));
    end
end

figure;
plot(t_vals, y_vals, 'LineWidth', 2);
xlabel('Time (s)');
ylabel('y(t)');
